About FGO Servant Database

What is this?

This is a simple database application for managing information about Servants from the mobile game Fate/Grand Order.

Features

  • View all servants in the database
  • See detailed information about each servant
  • Add new servants to the database
  • Edit existing servant information
  • Delete servants from the database

Technologies Used

  • Python Flask framework
  • SQLite database
  • HTML5 and CSS3
